A single static binary that discovers Squeezebox devices on the network
and writes their NVRAM configuration over the UDAP protocol. No daemon,
no GUI, no Lyrion Music Server dependency for setup.

## Install

```bash
brew install yo61/tap/go-udap
```

Or download a release binary from
[GitHub releases](https://github.com/yo61/go-udap/releases/latest) —
macOS, Linux, and Windows builds for amd64 and arm64.
