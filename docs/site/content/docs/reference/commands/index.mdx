# Commands

| Command | Purpose |
|---|---|
| [`discover`](./discover.mdx) | Discover devices on the network |
| [`info`](./info.mdx) | Show metadata for one device |
| [`read`](./read.mdx) | Read NVRAM parameters from a device |
| [`get`](./get.mdx) | Read specific parameters by name |
| [`set`](./set.mdx) | Write NVRAM parameters |
| [`reboot`](./reboot.mdx) | Reboot the device |
| [`getip`](./getip.mdx) | Query the device's IP / subnet / gateway |
| [`interfaces`](./interfaces.mdx) | List host network interfaces usable for discovery |

Global flags that apply to every command are documented under
[Global flags](../global-flags.mdx).
