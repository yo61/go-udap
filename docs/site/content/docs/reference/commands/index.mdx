# Commands

| Command | Purpose |
|---|---|
| [`discover`](./discover) | Discover devices on the network |
| [`info`](./info) | Show metadata for one device |
| [`read`](./read) | Read NVRAM parameters from a device |
| [`get`](./get) | Read specific parameters by name |
| [`set`](./set) | Write NVRAM parameters |
| [`reboot`](./reboot) | Reboot the device |
| [`getip`](./getip) | Query the device's IP / subnet / gateway |
| [`interfaces`](./interfaces) | List host network interfaces usable for discovery |

Global flags that apply to every command are documented under
[Global flags](../global-flags).
