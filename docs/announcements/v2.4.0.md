# go-udap v2.4.0 is available

[Get it on the GitHub release page](https://github.com/yo61/go-udap/releases/tag/v2.4.0) — or `brew upgrade go-udap` if you installed via Homebrew.

## What's new since v2.3.0

**Man pages.** Release archives now ship `go-udap.1` plus one page per subcommand — `man go-udap`, `man go-udap-set`, and so on. They're generated from the command tree, so flags and defaults stay in lockstep with the binary. Homebrew users get them registered on the next `brew upgrade`; from a tarball, drop `share/man/man1/*.1` anywhere on your `MANPATH`.

**Refreshed README and [docs landing](https://yo61.github.io/go-udap/).** The opening paragraph now states plainly what go-udap is and what it replaces — one static binary, no Lyrion Music Server required for setup, no Perl, no GUI — followed by a feature list and a realistic quick-start session.
